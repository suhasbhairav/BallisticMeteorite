Ballistic Meteorite.

Ballistic Meteorite is an IDE for python. The source code is in Java. It is simple and easy to navigate.

Requirements

Java - 1.7 Ubuntu - 14.04 Python - default version on Ubuntu (tested on 2.7)

The above versions are the ones on which it has been tested. There are very high chances of Ballistic Meteorite working on lower versions as well.

How to execute

Executing Ballistic Meteorite is not too difficult. It contains an executable jar which can be used to launch the IDE.
